# Lyrics AI Bot

Telegram-бот для поиска текстов песен и работы с ними прямо в чате.

Можно отправить название песни, выбрать нужный вариант и получить текст. После этого бот запоминает песню и понимает запросы вроде «переведи эту песню», «объясни смысл» или «что значит эта строка?».

## Что умеет бот

* искать песню по названию и исполнителю;
* показывать несколько вариантов при совпадениях;
* выводить полный текст песни;
* переводить выбранную песню;
* объяснять общий смысл;
* разбирать отдельные строки;
* понимать обычные сообщения без строгих команд;
* запоминать выбранную песню в рамках сессии;
* удалять сообщения текущего диалога.

Примеры запросов:

```text
Numb Linkin Park
найди текст The Emptiness Machine
переведи эту песню
переведи на английский
объясни смысл песни
что значит фраза «I've become so numb»
переведи: I tried so hard and got so far
```

## Команды

| Команда              | Описание                  |
| -------------------- | ------------------------- |
| `/start`             | Запуск и краткая справка  |
| `/search <название>` | Поиск песни               |
| `/current`           | Текущая выбранная песня   |
| `/translate`         | Перевод песни             |
| `/translate <язык>`  | Перевод на указанный язык |
| `/explain`           | Объяснение смысла         |
| `/clear`             | Очистка сообщений диалога |
| `/help`              | Примеры запросов          |

Большинство действий можно выполнять без команд.

## Требования

* Node.js 22+
* Telegram-бот от [@BotFather](https://t.me/BotFather)
* ключ OpenAI API

## Запуск

Установи зависимости:

```bash
npm install
```

Создай `.env`:

```bash
cp .env.example .env
```

Заполни переменные:

```env
TELEGRAM_BOT_TOKEN=...
OPENAI_API_KEY=...

OPENAI_ROUTER_MODEL=gpt-5.4-nano
OPENAI_TEXT_MODEL=gpt-5.4-mini
```

Запуск:

```bash
npm run dev
```

или

```bash
npm start
```

## Развёртывание

Можно запустить на Railway как обычный сервис.

Добавь переменные окружения:

```env
TELEGRAM_BOT_TOKEN=...
OPENAI_API_KEY=...
OPENAI_ROUTER_MODEL=gpt-5.4-nano
OPENAI_TEXT_MODEL=gpt-5.4-mini
NODE_ENV=production
```

Команда запуска:

```bash
npm start
```

## Ограничения

* тексты зависят от базы LRCLIB;
* после перезапуска выбранная песня сбрасывается;
* старые сообщения могут не удаляться;
* тексты песен защищены авторским правом;
* стоимость OpenAI зависит от количества запросов.
